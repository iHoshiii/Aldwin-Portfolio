# Modern Portfolio

A clean, simple, and modern portfolio built with **React**, **Node.js**, and deployed on **Vercel**.

## 🚀 Features

- **Modern UI**: Clean and minimalist design with smooth animations.
- **Project Showcase**: 6 featured projects with high-quality previews.
- **Responsive**: Fully optimized for mobile, tablet, and desktop.
- **Vercel Deployment**: Optimized for fast delivery via Vercel's global edge network.

## 🛠️ Tech Stack

- **Frontend**: React.js, CSS3 (Vanilla)
- **Icons**: Lucide React
- **Hosting**: Vercel

## 📁 Project Structure

```text
/src
  /components     # Reusable UI components
  /assets         # Static assets (images, fonts, etc.)
  /styles         # Global CSS styles
App.jsx           # Main application logic
main.jsx          # Entry point
```

## 🛠️ Installation & Setup

1. **Clone the repository**
   ```bash
   git clone <your-repository-url>
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Run development server**
   ```bash
   npm run dev
   ```

4. **Build for production**
   ```bash
   npm run build
   ```

## 🌐 Deployment

This project is configured for deployment on [Vercel](https://vercel.com). Simply link your GitHub repository to Vercel for automatic deployments.
